How to mine Pirate with a GPU or ASIC miner.

GPU MiningTOP

Firstly, you will need to decide what GPU mining software you are going to use. There are many available to choose from. You will need to experiment to find which one works best for you. DYOR on this and be sure to virus scan all the things! This pool takes no responsibility for 3rd party software.

You are only able to mine Pirate using a zS address.

You are only able to mine Komodo with an R address.

If you mistype, or input an invalid address, you will not receive any funds and the pool will be unable to retrieve these. Please ensure you have copied your zSaddress correctly.

DO NOT, I repeat, DO NOT mine to an exchange wallet. This will cause all sorts of problems for the exchange to deal with, most likely leading to closed withdrawals and deposits as we have seen before. Refer to the information at the bottom of the page to get an Agama wallet.

Use the config examples below, and if you have multiple mining rigs you can add a .minername at the end of your username(wallet addr) to see stats broken down by rig on the worker stats page.

Here are some setup configurations for a couple of well-known GPU miners. All you will have to do is choose the correct port and change your username to your R(Komodo) or zS(Pirate) address.

EWBF's Cuda Equihash Miner
./miner --server {{=it.portalConfig.website.stratumHost}} --port {{= Object.keys(it.poolsConfigs[Object.keys(it.poolsConfigs)[0]].ports)[0] }} --user zBMzwbMsdkfjh...H82hfLjsoie.Miner1 --pass x --fee 0 --pec
dstm's ZCash / Equihash Nvidia Miner
./zm --server {{=it.portalConfig.website.stratumHost}} --port {{= Object.keys(it.poolsConfigs[Object.keys(it.poolsConfigs)[0]].ports)[0] }} --user zBMzwbMsdkfjh...H82hfLjsoie.Miner1
You can also look into using the miner below:
Bminer for CUDA GPU's Claymore's ZCash/BTG AMD GPU Miner

While it is possible to CPU mine, this pool does not support doing so on PIRATE.

Generate a PIRATE Wallet and AddressTOP
Currently to have a Pirate wallet, you must download the blockchain. Which means running Agama in NATIVE mode, as well as syncing the entire blockchain. DO NOT bypass this by mining to exchanges. This will cause severe problems for the exchange.

You can use any of the following wallets:
Pirate Wallets
Agama Multi-Currency Wallet
PirateOcean QT for Windows & Mac
Treasure Chest Wallet
Thanks to @kaks for making this tutorial for running Agama Wallet.

If you use a CLI wallet. You can follow the instructions below:
  1. Build Komodo
  2. Launch PIRATE chain and wait for it to sync fully: ./komodod -ac_name=PIRATE -ac_supply=0 -ac_reward=25600000000 -ac_halving=77777 -ac_private=1 -addnode=136.243.102.225
  3. Generate a Z address: ./komodo-cli -ac_name=PIRATE z_getnewaddress
  4. Securely backup your private key (keep this private key safe): ./komodo-cli -ac_name=PIRATE z_exportkey "zaddr"

Handy CLI commands
Check your balance: ./komodo-cli -ac_name=PIRATE z_gettotalbalance
Send your coins: ./komodo-cli -ac_name=PIRATE z_sendmany "fromZSaddress" '[{"address": "toZSaddress" ,"amount": 1000}]'
Generate a new Sapling address(starts with zS) ./komodo-cli -ac_name=PIRATE z_getnewaddress
Create a specific sized UTXO in your wallet. ./komodo-cli -ac_name=PIRATE z_sendmany "zaddr1" '[{"address":"zaddr1","amount": 1000}]'
